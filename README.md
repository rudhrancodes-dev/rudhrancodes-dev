### [Aegis](https://github.com/rudhrancodes-dev/Aegis)
Real-time network anomaly detection system using Isolation Forest machine learning with a live security-focused interface.

### [revfiresecurity](https://github.com/rudhrancodes-dev/revfiresecurity)
Premium static business website for Rev Fire & Security Solutions built with HTML, CSS, JavaScript, Three.js, and custom SVG branding.
